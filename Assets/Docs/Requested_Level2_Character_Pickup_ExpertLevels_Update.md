# Requested Gameplay Update

Changes included in this version:

1. Level 2 routing was redesigned so the beam must go through two mirrors. A central laser blocker prevents the one-mirror shortcut.
2. Character visuals were rotated so the player no longer appears to walk backward.
3. Pickup visuals now match their function:
   - Security keycard: flat blue/gold card
   - Energy core: glowing energy sphere
   - EMP device: small sci-fi blaster
   - EMP charge: glowing battery cell
4. Two expert levels were added:
   - Level05: five-mirror sealed crypt maze with item requirements
   - Level06: six-mirror final catacomb with more item requirements and tougher routing

Recommended commit:
`git add . && git commit -m "Fix level two route characters pickups and add expert levels"`
