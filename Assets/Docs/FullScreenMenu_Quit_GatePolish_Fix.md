# Full Screen Menu, Quit Button, and Gate Polish Fix

Fixes:
1. Main menu is now full-screen and no longer uses a small floating panel.
2. Main menu text is separated from buttons, so text should not be hidden or overlapped.
3. Quit button now works in Unity Editor by stopping Play Mode, and in builds by calling Application.Quit.
4. Exit gate is more polished:
   - Larger stone side pillars.
   - Top beam, lower lip, golden lintel, and cap blocks.
   - Wooden door panel with metal vertical bars, cross bars, and handle.
   - Door decorations are parented to SecurityDoor, so they move with the door.
