# Cartoon UI and Reflection Counter Update

Changes:
1. UI prompt boxes now use a stronger cartoon-style font preference.
2. Objective text, pickup messages, mirror prompts, rail prompts, switch prompts, and EMP prompt support word wrapping.
3. Objective prompt height is calculated automatically, so long instructions should no longer overflow.
4. Top-right counter uses the same cartoon UI style.
5. Top-right counter now includes Reflections: current / required.
6. LightReflection exposes CurrentMirrorReflections and RequiredMirrorReflections for the UI.
