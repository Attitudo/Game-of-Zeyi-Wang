# Rail Mirror Start Offset and Decoy Mirror Update

Changes:
1. Rail mirrors now start away from their correct solution point.
2. The rail no longer turns green or reveals the correct position.
3. Later levels include extra decoy mirrors:
   - Level 3: one decoy mirror
   - Level 4: two decoy mirrors
   - Level 5: three decoy mirrors
   - Level 6: four decoy mirrors
4. Decoy mirrors are also movable/rotatable so they look plausible, but they are not required for the intended solution.

Controls:
- Q / E: rotate mirror
- Z / C: slide mirror along its rail
- X: toggle laser from Level 2 onward
