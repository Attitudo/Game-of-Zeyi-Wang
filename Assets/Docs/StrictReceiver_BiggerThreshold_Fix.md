# Strict Receiver and Bigger Door Threshold Fix

Fixes:
1. Receiver activation is now strict.
   - The laser must actually point at / hit the receiver.
   - The old wide near-beam tolerance was reduced from 1.85 to 0.28.
   - This prevents the door from opening when the beam only passes nearby.
2. Door threshold is larger.
   - Lower stone lip is thicker and wider.
   - A front stone step is added outside the gate.
3. Existing gate polish is kept.
